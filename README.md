# CSE412_Project
The Jewellery E-Commerce Management System will be a web-based platform consisting of two primary components: a customer-facing storefront and an administrative management dashboard. 

# JOHROT E-Commerce Website
A jewelry e-commerce website built with WordPress, WooCommerce, and Elementor.

## Tech Stack
- WordPress (CMS)
- WooCommerce (E-commerce)
- Elementor (Page Builder)

## Features
- Product categories (Rings, Necklaces, Earrings, etc.)
- Shopping cart and checkout
- Cash on delivery payment
- Delivery zones with shipping charges
- Mobile responsive design
